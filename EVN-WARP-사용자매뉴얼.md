## 1. 시스템 개요

**EVN Warp**는 이브이앤솔루션의 전사 목표(전략과제)를 팀 단위 실행계획으로 연결하고, 주간 진행 상황을 추적·공유하기 위한 내부 목표관리시스템입니다.

### 핵심 개념

| 개념 | 설명 |
|------|------|
| **전략과제 (A3)** | 회사 전략을 구현하는 과제 단위. 전략 A(확장과 성장) / 전략 B(AI 기반 조직운영)로 구분 |
| **전략테마 (상위과제)** | 경영진이 등록하는 최상위 방향성 과제. 팀 실행과제들의 상위 항목 |
| **실행과제 (하위과제)** | 각 팀이 담당하는 실제 실행 단위. 반드시 전략테마에 연결됨 |
| **실행안** | 하나의 실행과제를 구성하는 세부 일정 항목. 3페이지 간트차트에 막대로 표시 |
| **업무활동** | 회의·출장·보고 등 팀원이 수행하는 일상 업무 기록 (4페이지 입력) |

### 전략 구분

- **전략 A** — 확장과 성장 (남색/Indigo 계열)
- **전략 B** — AI 기반 조직운영 (보라/Violet 계열)

### 과제 등록 주체

| 구분 | 등록 주체 | 설명 |
|------|-----------|------|
| 전략테마 (상위과제) | 경영진 | 전략 A/B의 대분류. 팀이 없는 최상위 항목 |
| 실행과제 (하위과제) | 각 팀 담당자 | 전략테마 아래 연결된 팀별 실행 과제 |

## 2. 화면 구성

좌측 사이드바를 통해 4개 페이지로 이동합니다.

```
┌─────────────────────────────────────────────┐
│  1페이지  경영 대시보드  /dashboard           │  ← 경영진 현황 파악
│  2페이지  전략과제 A3   /a3                  │  ← 과제 등록 및 관리
│  3페이지  주간업무      /weekly               │  ← 간트 + 주간 집계
│  4페이지  업무공간      /notes                │  ← 업무활동 기록
└─────────────────────────────────────────────┘
```

각 페이지 헤더에는 **주/월 탐색 버튼(← →)** 이 있어 이전·다음 주(또는 월)의 데이터를 조회할 수 있습니다.

---

## 3. 1페이지 — 경영 대시보드

**접속 경로**: `/dashboard`  
**목적**: 경영진이 전사 KPI 달성 현황, 전략과제 실행 상태, 주요 공지를 한 화면에서 파악

### 화면 구성 (3컬럼)

```
┌─────────────────┬──────────────────┬─────────────────┐
│  ① 전사 KPI    │  ② 전략과제 현황  │  ③ 알림마당      │
│  달성 현황      │                  │                 │
└─────────────────┴──────────────────┴─────────────────┘
```

---

### ① 전사 KPI 달성 현황

연간 KPI 목표 대비 월별 달성률을 차트로 표시합니다.

**KPI 값 입력 방법**

1. 헤더 우상단 **"KPI 입력"** 버튼 클릭
2. 슬라이드 패널에서 KPI 항목별 해당 월 실적값 입력
3. **저장** 클릭

> 전사 KPI 항목(목표치·단위·카테고리)은 시스템 관리자가 사전에 등록합니다.  
> 여기서 입력하는 실적은 **전사 차원의 월별 수치**이며, 각 과제별 KPI와는 별개입니다.

---

### ② 전략과제 현황

선택한 주의 팀별 과제 진행 상태를 요약합니다.

- **상태 4칸 요약**: 정상 / 지연 / 조치필요 / 완료 건수
- **팀별 테이블**: 각 과제 코드 배지(색상으로 상태 표시) + 팀별 카운트
- **주 탐색**: 헤더 우측 `← 이번 주 →` 버튼으로 다른 주 조회

> 이 섹션의 데이터는 **3페이지 간트탭**에서 팀원이 상태 버튼을 클릭하면 자동 반영됩니다. 별도 입력 불필요.

---

### ③ 알림마당

4페이지(업무공간)에서 `@멘션`을 포함한 활동 기록이 자동으로 이곳에 표시됩니다.

| 구분 | 조건 | 표시 방식 |
|------|------|-----------|
| **전체공지** | 멘션 필드에 `@전체` 또는 `@all` 포함 | 황색 배경, 최상단 |
| **멘션** | 그 외 멘션 값이 있는 항목 | 일반 리스트 |

- 항목 클릭 시 해당 활동의 수정 페이지(4페이지)로 이동
- **입력 방법**: 4페이지 → 활동 추가 시 멘션 필드에 `@전체`, `@홍길동` 등 입력

## 4. 2페이지 — 전략과제 A3

**접속 경로**: `/a3`  
**목적**: 전략과제 등록, 실행계획(실행안) 관리, A3 문서 조회

### 목록 화면 구성

- **팀별 섹션**으로 과제 그룹 표시
- 각 섹션: 진행 중 과제 → 완료 과제(흐리게) → 중단 과제(주황) 순서
- 섹션 헤더에 진행/완료/중단 건수 배지 표시

```
[팀명]  진행 3  완료 1  ────────────────────── 4건

[A] EVN-A1-01  과제명                팀명  오너 홍길동  진행중  D-45  >
    ├── EVN-A1-01-01  하위과제 1     팀명  오너 김철수  진행중  D-20  >
    └── EVN-A1-01-02  하위과제 2     팀명  오너 이영희  완료    D+3   >
```

---

### 새 과제 등록

우상단 **"새 과제 등록"** 버튼 클릭 → 등록 폼은 아래 4개 섹션으로 구성됩니다.

#### 기본 정보

| 필드 | 설명 | 필수 |
|------|------|------|
| **상위 전략과제** | 전략테마 A 또는 B 중 연결할 상위 과제 선택 | ✓ |
| **담당 팀** | 드롭다운에서 팀 선택 | ✓ |
| **과제명** | 과제 제목 | ✓ |
| **과제 오너** | 과제 책임자 이름 | ✓ |
| **상태** | 진행중 / 완료 / 보류 / 지연 | ✓ |
| **시작일 / 목표 기한** | 과제 전체 기간 | ✓ |
| **위원회 확정** | 과제가 최종 확정되었으면 체크 | |

> 과제 코드(예: EVN-A1-01)는 저장 시 자동 부여됩니다.

#### KPI 설정

과제 고유의 성과지표를 설정합니다. **정량** 또는 **정성** 중 선택.

| KPI 유형 | 세부 유형 | 설명 |
|----------|-----------|------|
| **정량** | 금액 / 일자 / 인원수 / 건수 / 대수 / 비율 / 기타 | 수치로 측정 가능한 목표 |
| **정성** | — | 달성 내용을 텍스트로 기술 |

- **+ KPI 추가** 버튼으로 여러 항목 등록 가능
- 정량 KPI 중 **금액** 항목은 경영 대시보드 전사 KPI와는 별개로 과제 상세 페이지에서 확인

#### 1. 문제와 목표

과제의 배경과 목표를 A3 방식으로 서술합니다.

| 필드 | 설명 |
|------|------|
| **문제 (정량)** | 현재 문제 상황을 수치 중심으로 기술 |
| **달성 목표 (정량)** | 과제 완료 시 달성할 목표를 수치로 기술 |

#### 2. 대책과 실행안

과제의 세부 실행 계획을 입력합니다.

| 필드 | 설명 |
|------|------|
| **실행안 내용** | 세부 실행 항목 설명 (번호 자동 부여) |
| **시작일 / 완료일** | 해당 실행안의 기간 |

- **+ 실행안 추가** 버튼으로 여러 항목 등록
- 기간을 입력하면 폼 하단에 **간트 미리보기** 자동 표시
- 저장 후 3페이지 간트차트에 막대로 표시됨

#### 3. 월별 목표 및 리소스 (선택)

**"+ 월별 목표 추가"** 버튼을 눌러 펼친 후 입력합니다.

| 항목 | 설명 |
|------|------|
| **매출 (만원)** | 월별 매출 목표 |
| **예산 (만원)** | 월별 소요 예산 |
| **인력 (명)** | 월별 필요 인원 |

---

### 과제 상세 페이지

목록에서 과제를 클릭하면 A3 상세 페이지로 이동합니다.

**상단 버튼**

| 버튼 | 기능 |
|------|------|
| **과제정의서 PPT** | A3 내용을 PPT 형식으로 다운로드 |
| **수정** | 과제 정보 편집 폼으로 이동 |
| **중단 / 재개** | 과제 중단 처리 (중단 사유 입력) 또는 재개 |

**상세 페이지 표시 내용**

- 기본 정보 (팀, 오너, 기간, 상태, D-day)
- KPI 목표치 카드
- 하부 과제 목록 + **"하부 과제 추가"** 버튼 (상위과제에서만 표시)
- 1. 문제와 목표 서술
- 2. 대책과 실행안 (과제 전체 기간 기준 간트)
- 3. 월별 목표 및 리소스 표

---

### 과제 중단 처리

- 상세 페이지 우상단 **"중단"** 버튼 클릭 → 중단 사유 입력
- 목록에서 주황색 카드로 분리 표시
- 재개하려면 중단된 과제 클릭 → **"재개"** 버튼 클릭

## 5. 3페이지 — 주간업무

**접속 경로**: `/weekly`  
**목적**: 팀별 실행계획 간트 확인, 주간 진행 상태 입력, 이번 주·차주 업무 자동 집계

### 탭 구성

| 탭 | 내용 |
|----|------|
| **간트** | 8주 윈도우 실행계획 간트차트 + 주간 상태 입력 |
| **주간 관리** | 이번 주 완료 / 차주 계획 자동 집계 |

---

### 간트 탭

**화면 구성**

```
팀명 ─────────────────────────────────────────────────
[A] EVN-A1-01  [●●●●]     W25  W26  W27  W28 ...
    과제명  [▼2]          │    ████ (실행안 막대)  ◇
    ├ ① 실행안 내용        │         ██████
    └ ② 실행안 내용        │                 │(오늘선)
```

**간트 구성 요소**

| 표시 | 의미 |
|------|------|
| 색상 막대 | 실행안 기간 |
| ◇ 다이아몬드 | 과제 마감일 |
| `← 완료` / `예정 →` | 8주 윈도우 밖의 과제 |
| 빨간 세로선 | 오늘 날짜 |
| 파란 배경 컬럼 | 이번 주 (현재 선택 주) |
| 하늘색 배경 컬럼 | 차주 |
| `▼숫자` 버튼 | 실행안 펼치기 / `▲` 접기 |

**주간 상태 입력 방법** (매주 핵심 액션)

각 과제 행 우측 상단의 **상태 버튼(● ● ● ●)** 클릭:

| 버튼 색상 | 의미 |
|-----------|------|
| 🟢 초록 | 정상 |
| 🟡 황색 | 지연 |
| 🔴 빨강 | 조치필요 |
| 🔵 파랑 | 완료 |

- 클릭 즉시 저장 (별도 저장 버튼 없음)
- 선택한 상태가 **1페이지 대시보드 → 전략과제 현황**에 즉시 반영

**실행안 펼치기**

- 과제 행의 `▼숫자` 버튼 클릭 → 실행안 막대를 개별 행으로 펼쳐서 확인
- 다시 클릭하면 `▲` 접기

---

### 주간 관리 탭

이번 주(Weekly Completed Works)와 차주(Weekly Planned Works)를 좌우로 나란히 자동 집계합니다.

**자동 집계 조건**

| 섹션 | 표시 내용 |
|------|-----------|
| 이번 주 완료 | 이번 주 기간에 걸친 실행안 + 4페이지에서 이번 주에 등록한 업무활동 |
| 차주 계획 | 차주 기간에 걸친 실행안 + 4페이지에서 차주 날짜로 등록한 업무활동 |

> 별도 입력 없이 **2페이지 실행안**과 **4페이지 업무활동**을 충실히 등록하면 자동으로 채워집니다.

## 6. 4페이지 — 업무공간

**접속 경로**: `/notes`  
**목적**: 팀원의 일상 업무활동(회의·출장·보고 등) 기록 및 협업 알림 허브

### 탭 구성

| 탭 | 내용 |
|----|------|
| **캘린더** | 전통 월간 캘린더 (7열 × N주) — 날짜별 활동 한눈에 보기 |
| **업무노트** | 회의록 / 출장보고 / 업무협조 섹션별 목록 |

---

### 캘린더 탭

- **월 탐색**: 헤더 `← 년/월 →` 버튼으로 이전·다음 월 이동
- **활동 추가**: 날짜 셀에 마우스를 올리면 나타나는 **+** 버튼 클릭
- **활동 수정/삭제**: 캘린더 셀 안의 활동 칩(색상 태그) 클릭
- 오늘 날짜는 빨간 원으로 강조, 일요일/토요일은 배경색 구분

---

### 업무노트 탭

해당 월의 활동을 유형별로 분류하여 목록으로 표시합니다.

| 섹션 | 포함 유형 | 빠른 추가 버튼 |
|------|-----------|----------------|
| **회의록** | 내부회의, 외부회의 | 우측 + 내부회의 / + 외부회의 |
| **출장보고** | 국내출장, 해외출장 | 우측 + 국내출장 / + 해외출장 |
| **업무협조** | 멘션(@)이 포함된 모든 유형 | — |

---

### 활동 추가 방법

| 경로 | 방법 |
|------|------|
| 캘린더 탭 | 날짜 셀 hover → + 버튼 클릭 (날짜 자동 입력) |
| 헤더 버튼 | 우상단 **"활동 추가"** 버튼 클릭 |
| 업무노트 탭 | 각 섹션 우측 유형별 버튼 클릭 (유형 자동 선택) |

**입력 항목**

| 필드 | 설명 | 필수 |
|------|------|------|
| **활동 유형** | 아래 9가지 중 선택 | ✓ |
| **날짜** | 활동 날짜 (YYYY-MM-DD) | ✓ |
| **연결 과제** | 어떤 전략과제와 연관된 활동인지 선택 | ✓ |
| **담당 팀** | 팀 선택 | ✓ |
| **제목** | 활동 제목 | ✓ |
| **내용** | 상세 내용 (자유 텍스트) | |
| **@멘션** | 관련 인원 또는 `@전체` 입력 시 1페이지 알림마당에 표시 | |

**활동 유형 9가지**

| 유형 | 색상 | 주요 용도 |
|------|------|-----------|
| 내부회의 | 파랑 | 사내 회의, 부서 간 회의 |
| 외부회의 | 보라 | 고객사, 파트너사 미팅 |
| 국내출장 | 주황 | 국내 방문·출장 |
| 해외출장 | 빨강 | 해외 방문·출장 |
| 발표/보고 | 청록 | 경영진 보고, 발표 자료 |
| 교육/연수 | 황색 | 내부·외부 교육 참가 |
| 전시/행사 | 장미 | 전시회, 행사 참가 |
| 사무업무 | 회색 | 기타 일반 업무 |
| 실적추가 | 에메랄드 | KPI 관련 실적 메모 |

### @멘션 활용법

| 입력 값 | 효과 |
|---------|------|
| `@전체` 또는 `@all` | 1페이지 알림마당 → **전체공지** 섹션에 황색 배경으로 표시 |
| `@홍길동` 등 특정 인원 | 1페이지 알림마당 → **멘션** 섹션에 표시 |
| (멘션 있는 항목, 이번 주 날짜) | 3페이지 주간관리 탭 → 해당 과제 완료사항에도 자동 표시 |

## 7. KPI 구조 이해

시스템 내 KPI는 세 가지 레벨로 구분됩니다. 혼동하지 않도록 주의하세요.

| 레벨 | 명칭 | 입력 위치 | 용도 |
|------|------|-----------|------|
| **전사 KPI** | 회사 전체 연간 목표 | 1페이지 대시보드 → KPI 입력 버튼 | 매출·영업이익 등 전사 지표 월별 실적 관리 |
| **과제 KPI 목표** | 개별 과제의 성과지표 | 2페이지 A3 등록 폼 → KPI 설정 | 과제 완료 기준이 되는 정량/정성 목표 |

> 두 KPI는 서로 다른 데이터로 연동되지 않습니다. 전사 KPI는 회사 전체 수치를, 과제 KPI는 해당 과제 단위의 목표를 각각 독립적으로 관리합니다.

## 8. 데이터 흐름 한눈에 보기

```
  [2페이지 A3]                      [4페이지 업무공간]
  ① 전략테마 등록 (경영진)           ③ 업무활동 기록
  ② 실행과제 등록 (각 팀)               날짜 / 유형 / @멘션
     └ 실행안 (기간 설정)
          │                                  │
          ▼                                  ▼
  [3페이지 주간업무]                  [1페이지 대시보드]
  간트탭 → 상태 클릭 ──────────────→ 전략과제 현황 자동 집계
  주간관리탭 → 실행안+활동 자동 집계   @멘션 → 알림마당 자동 표시
```

### 주간 루틴 권장 순서

| 시점 | 담당 | 액션 |
|------|------|------|
| **과제 시작 시** | 팀 담당자 | 2페이지 → 실행과제 + 실행안 등록 |
| **매주 월요일** | 팀원 | 3페이지 간트탭 → 이번 주 과제 상태 클릭 (정상/지연/조치필요) |
| **수시** | 팀원 | 4페이지 업무공간 → 회의·출장·보고 기록 등록 |
| **매주 금요일** | 팀장·경영진 | 3페이지 주간관리탭 → 완료사항/계획사항 확인 |
| **수시** | 경영진 | 1페이지 대시보드 → 전사 현황 및 알림마당 조회 |
